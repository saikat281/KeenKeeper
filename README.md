# Project Name:  KeenKeeper – Personal Friendship Tracker

## 📝 Description
KeenKeeper is a responsive web application that helps users maintain and manage their personal Friendship info. It allows users to track interactions with friends, monitor communication goals, and view a timeline of calls, texts, and video check-ins to stay connected consistently.

---

## ⚙️ Technologies Used
- React.js  
- React Router 
- Tailwind CSS / Daisy UI 
- Recharts (for analytics visualization)  
- JSON (for mock data)

---

##  Key Features
### 1. 👥 Friend Management Dashboard
View all friends with status indicators, tags, and last contact details.

### 2. 📜 Interactive Timeline Tracking
Log calls, texts, and video interactions instantly from the Friend Details page with real-time timeline updates.

### 3. 📊 Friendship Analytics
Visualize communication patterns using a pie chart that shows the distribution of Call, Text, and Video interactions.
